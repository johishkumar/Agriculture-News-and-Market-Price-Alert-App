## рҹҡҖ Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/uzhavar.git
```

### Navigate to Project

```bash
cd uzhavar
```

### Install Dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd server
npm install
```

### Start Development Server

Frontend

```bash
npm start
```

Backend

```bash
npm run dev
```
